# CSS 方案指纹

> style_layer 插件知识库。涵盖主流 CSS 方案的识别签名。

---

| 方案 | Class 命名特征 |
|:---|:---|
| Tailwind CSS | 原子类如 `flex`, `pt-4`, `bg-blue-500` |
| Styled-components | `sc-` 前缀 |
| Emotion | `css-` 前缀 |
| CSS Modules | `filename_classname_hash` 格式 |
| UnoCSS | 类似 Tailwind 的原子类 |
| Vanilla CSS | 语义化类名，无特殊模式 |
