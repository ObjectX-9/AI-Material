# 性能评估知识

> performance_layer 插件知识库。涵盖 Lighthouse 评分标准和 Core Web Vitals 阈值。

---

## Core Web Vitals 阈值

| 指标 | 🟢 Good | 🟡 Needs Improvement | 🔴 Poor |
|:---|:---|:---|:---|
| LCP | ≤ 2.5s | ≤ 4.0s | > 4.0s |
| INP | ≤ 200ms | ≤ 500ms | > 500ms |
| CLS | ≤ 0.1 | ≤ 0.25 | > 0.25 |

## Lighthouse 评分等级

| 等级 | 分数范围 |
|:---|:---|
| 🟢 Good | 90–100 |
| 🟡 Average | 50–89 |
| 🔴 Poor | 0–49 |
